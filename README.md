## 📋 About The Project

**Kohinoor Gemstone** is a full-stack e-commerce platform for a premium gemstone business based in Bareilly, India. The platform features:

- 🌐 Modern responsive design
- 🤖 AI-powered gemstone recommendations
- 💬 Real-time customer chat
- 🔐 Secure authentication system
- 📱 Mobile-first approach
- 🎨 Elegant dark/light themes
